# MZ2POL - Management Zone to Policy Migration

Tools and guidance for migrating from Management Zones to Policy-based access control in Dynatrace.

## Structure
- NOTEBOOKS/ — Dynatrace notebook JSON files
- PDFs/ — Printable versions of each notebook
- markdown/ — Markdown exports of the notebooks

## Notebook Lineup
1. SDK MZ Analysis Tool — Analyze existing management zones
2. Introduction: Why Migrate — Benefits and overview
3. Access Control Model — Policies and access control concepts
4. Assessment & Planning — Migration assessment and planning
5. Policies and Boundaries — Defining policies and boundaries
6. Segments Implementation — Implementing segments
7. Migration Execution — Executing the migration
8. Validation & Troubleshooting — Validating and resolving issues

## Usage
1. Choose a format: import JSON from NOTEBOOKS/, read PDFs/ for print, or view markdown/ for lightweight browsing.
2. Use the SDK analysis tool first, then follow the numbered migration path.
