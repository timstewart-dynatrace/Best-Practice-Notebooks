# OPLOGS - OpenPipeline Logs

End-to-end guidance for managing logs with Dynatrace OpenPipeline.

## Structure
- NOTEBOOKS/ — Dynatrace notebook JSON files for each topic
- PDFs/ — Printable versions of the notebooks
- markdown/ — Markdown exports of the notebooks

## Notebook Lineup
1. Fundamentals — Core OpenPipeline concepts
2. Migration — Moving from other logging platforms
3. Pipeline Processing — Transformations and processing stages
4. Buckets & Governance — Storage management and governance
5. Querying & Parsing — Query techniques and parsing rules
6. Topology — Log topology and relationships
7. Analytics — Advanced log analytics
8. Security — Security, masking, and compliance

## Usage
1. Choose a format: import JSON from NOTEBOOKS/, read PDFs/ for print, or view markdown/ for lightweight browsing.
2. Start with Fundamentals before progressing to advanced topics.
